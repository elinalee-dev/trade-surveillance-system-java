# System Architecture

## Overview

The Trade Surveillance System is a full-stack web application designed to simulate financial transaction monitoring and cybersecurity risk detection workflows.

---

## Frontend

Technology:
- React
- JavaScript
- Recharts

Responsibilities:
- Display trade data
- Visualize alerts
- Show analytics dashboard

---

## Backend

Technology:
- Java Spring Boot
- REST APIs

Responsibilities:
- Process simulated trades
- Run anomaly detection logic
- Store and retrieve data

---

## Database

Technology:
- PostgreSQL

Responsibilities:
- Store trade records
- Store alerts
- Manage transaction history

---

## Detection Engine

The detection engine analyzes incoming trades for suspicious patterns such as:
- abnormal trading volume
- rapid transactions
- suspicious login activity
- unusual account behavior
