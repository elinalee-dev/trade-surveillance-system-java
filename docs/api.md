# Trade Surveillance System API Documentation

## Base URL

http://localhost:8080/api

---

# Trade Endpoints

## Get All Trades

GET /trades

Returns a list of all simulated trades stored in the system.

### Response Example

[
  {
    "id": 1,
    "accountId": "ACC1001",
    "symbol": "AAPL",
    "side": "BUY",
    "quantity": 1500,
    "price": 210.45,
    "ipAddress": "10.0.0.12"
  }
]

---

## Create Trade

POST /trades

Creates a new simulated trade and automatically runs anomaly detection logic.

### Request Body

{
  "accountId": "ACC2005",
  "symbol": "TSLA",
  "side": "SELL",
  "quantity": 12000,
  "price": 355.10,
  "ipAddress": "10.0.0.55"
}

---

# Alert Endpoints

## Get Alerts

GET /alerts

Returns suspicious activity alerts generated by the surveillance engine.

### Alert Types

- ABNORMAL_TRADE_VOLUME
- SUSPICIOUS_IP
- POTENTIAL_MARKET_MANIPULATION

### Response Example

[
  {
    "accountId": "ACC2005",
    "alertType": "ABNORMAL_TRADE_VOLUME",
    "severity": "HIGH",
    "message": "Trade quantity exceeded threshold: 12000"
  }
]
