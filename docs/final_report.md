# Final Report

## Project Summary

The Trade Surveillance System is a cybersecurity-focused financial monitoring platform designed to simulate real-world market surveillance workflows used in financial institutions.

The project combines:
- cybersecurity concepts
- anomaly detection
- transaction monitoring
- full-stack software development

---

## Objectives

- Detect suspicious trading activity
- Simulate surveillance workflows
- Visualize financial anomalies
- Practice backend API development

---

## Technologies Used

- Java
- Spring Boot
- PostgreSQL
- React
- REST APIs

---

## Key Features

- Real-time trade monitoring
- Alert dashboard
- Detection rule engine
- Suspicious activity visualization

---

## Future Improvements

- Machine learning anomaly detection
- Real-time streaming data
- Authentication system
- SIEM integration
